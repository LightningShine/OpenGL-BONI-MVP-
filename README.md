# RAJAGP (клиент)

OpenGL-клиент телеметрии картинга: карта трека, машины в реальном времени, тайминги, PRO-режим (сектора, круги, события). Данные — с СOM-порта (приёмник SX1280/LoRa) или с **RAJAGP Track Server** по WebSocket (нативный WinHTTP, работает на x64 и ARM64).

## Зависимость: репозиторий сервера рядом

Клиент использует общую библиотеку протокола `rajagp_core` (заголовки `core/include/rajagp/`) из репозитория [RAJAGP-Server](https://github.com/LightningShine/RAJAGP-Server) **по относительному пути** — оба репозитория должны лежать в одной папке:

```
source/repos/
├── OpenGL-BONI-MVP-/     ← этот репозиторий (клиент, ветка ARM-dev)
└── RAJAGP Server/        ← сервер + rajagp_core (общий протокол)
```

Без соседнего каталога `RAJAGP Server` клиент не соберётся.

## Сборка

Visual Studio: открыть `OpenGL.sln`, конфигурация `Debug|x64` (или ARM64), Build. Бинарь: `x64\Debug\OpenGL.exe`.

## Подключение к серверу

Меню **Networking → Connect to Server** (Shift+C): адрес `IP:8080`, в поле пароля — токен (обычный или админский). Админ получает управление гонкой (меню **Race**: старт/финиш/автостоп, флаги) и раздачу доступов (**Networking → Accounts**: создание юзеров на 1 час/1 день/бессрочно, токены копируются кликом).

Горячие клавиши — **F1**. Результаты: **Ctrl+S** — сохранить в .txt, **Ctrl+P** — печать.
